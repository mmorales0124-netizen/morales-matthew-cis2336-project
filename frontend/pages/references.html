AI Prompts Used During Development

Claude (Anthropic) was used to help design and code the front end. The prompts below summarize the requests made during development, in the order they were given.

Prompt 01 — Homepage

"Create a professional homepage for ArtConnect with a logo/title, navigation menu, welcome message, description, featured artwork section, featured event section, and a developer contact section."

Prompt 02 — Visual direction

"I want a dystopian visual direction, goth like the Cure, emo like Underoath, and inspired by Bladee's artwork and aesthetic."

Prompt 03 — Layout adjustment

"Center all the text below the 'Enter the Gallery' button."

Prompt 04 — Bug fix

"I opened it and I don't see the ArtConnect title with color" — requested a fix so the glitch/chromatic aberration effect on the title was visible.

Prompt 05 — Gallery page

"Create a gallery page displaying at least six artworks, each with image, title, artist, category, and price or 'Not for Sale,' enhanced with JavaScript for image enlargement."

Prompt 06 — Gallery layout

"Make the gallery grid show three artworks in one row and three on the next."

Prompt 07 — FAQ page

"Create an FAQ page with at least five questions that expand and collapse using JavaScript, visually organized and easy to navigate."

Prompt 08 — Style tweak

"Make 'Frequently Asked Questions' white."

Prompt 09 — References page

"Create a references page containing image sources, multimedia sources, external resources used, and AI prompts used during development."

Prompt 10 — Room booking form

"Develop a simple room booking form page (5 rooms). Develop validation checks for each room's availability (will be handled by the backend later). The form should follow best practices for validating form fields." — placed on the Events page.

Prompt 11 — Events page

"Create an events page displaying at least four upcoming art events, each with title, date, location, description, and image. JavaScript should be used to display event details dynamically. Use random images."

Prompt 12 — References update

"Can you update my references page" — requested addition of the Lorem Picsum image source and this updated prompt log.
